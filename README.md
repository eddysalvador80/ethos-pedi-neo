# ETHOS Pedi-Neo · Ácido-base

Herramienta **docente** de interpretación ácido-base neonatal-pediátrica: abstención-primero, cada lectura cita su fuente, no inventa cifras. PWA instalable y offline, con auto-actualización al publicar.

**App:** https://eddysalvador80.github.io/ethos-pedi-neo/

De 0 a experto en una sola herramienta:
- **Ruta** — escalera guiada N1→N12 con maestría comprobada (etapas-llave de gate ciego) + examen final que certifica.
- **Aprender** — recuperación activa: cada tarjeta con su check, "dominado" por acierto.
- **Práctica** — casos reales resueltos por el mismo motor, por tiers de dificultad, con repaso espaciado.
- **Interpretar / Referencia** — el motor de 4 pasos (Carmody) + bibliografía verificada (81 refs).

Núcleo verificado: 28 casos-oro de autoverificación al arrancar + 50 tests de banderas clínicas.
